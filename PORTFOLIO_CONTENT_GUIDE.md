## 🔤 FONT PAIRING

### Recommended: **Sora + Inter**

**Display Font (Headings):** Sora
- Weight: 600, 700, 800
- Use for: H1, H2, main headings
- Why: Modern, geometric, highly readable, professional

**Body Font (Content):** Inter
- Weight: 400, 500, 600
- Use for: Paragraphs, descriptions, UI text
- Why: Excellent screen readability, professional, widely used in tech

### Alternative Font Pairs

**Option 2:** Outfit + Open Sans
- More casual but still professional
- Great for creative roles

**Option 3:** Manrope + Source Sans Pro
- Warm and approachable
- Good for startups

**Option 4:** Plus Jakarta Sans + Inter
- Very modern and clean
- Trending in 2024-2025

**✅ RECOMMENDED:** Sora + Inter (Best for tech companies)

## 💡 PORTFOLIO CONTENT WRITING TIPS

### DO's:
✅ Use action verbs (Built, Developed, Implemented, Optimized)  
✅ Quantify results (40% faster, 100+ users, 15+ vulnerabilities)  
✅ Keep sentences short and scannable  
✅ Focus on impact, not just features  
✅ Use consistent tense (past tense for completed work)  
✅ Proofread multiple times for typos  

### DON'Ts:
❌ Use first person ("I built..." → "Built...")  
❌ Write long paragraphs (recruiters scan, not read)  
❌ Include irrelevant personal information  
❌ Use jargon or acronyms without context  
❌ Make it too flashy or animated  
❌ Forget to update contact information
